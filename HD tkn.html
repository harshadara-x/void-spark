  1. Installed Solana CLI and created a Devnet wallet.
  2. Requested SOL airdrop for transaction fees using `solana airdrop 2`.
  3. Installed SPL Token CLI using `cargo install spl-token-cli`.
  4. Created the token with `spl-token create-token`.
  5. Created an associated token account using `spl-token create-account TOKEN_ADDRESS`.
  6. Minted tokens to wallet using `spl-token mint TOKEN_ADDRESS AMOUNT`.
